# MoodCoin

## 개요
사용자의 메신저 대화 내용을 통해 사용자의 감정을 분석, 이를 기반으로 하여 자동으로 적금을 도와 시발비용의 낭비를 제어해주며, 감정 변화에 따른 솔루션 제공 (ex. 슬플 때 듣는 음악, 감정 추스리는 법 등)
